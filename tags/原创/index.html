Git拉取远程分支到本地

介绍 很多人不知道如何拉取远程分支到本地，比如刚刚clone下一个新库，但是却只有master,没有开发分支develop，以下介绍如何拉取远程分支到本地。 方法 在本地创建同名分支 git checkout -b dev(本地分支名称) origin/dev(远程分支名称) 拉取远程内容 git pull origin dev(远程分支名称)
