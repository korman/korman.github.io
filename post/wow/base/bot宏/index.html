必看！bot宏

2020年9月15日
| 魔兽世界

基础

机器人用“停止”命令后，鼠标右键点击机器人即可对话，可以分配机器人职责（坦克、奶、输出——如“坦克”选择“近战，坦克”；“治疗”选择“远程，治疗”、或“近战，治疗”，或：“治疗”），交易装备给他；交易完成用“跟随”命令即可

机器人装备指南

伤害受玩家装备等级和他自己装备影响，好装备对其影响很大，尤其是武器 貌似可以享受装备属性影响（暴击、法伤、攻强、防御、躲闪等） 可以触发武器附带技能，不能触发饰品附带技能 穿装备后会刷新装备原始属性，如变回装绑，附魔、宝石消失。

坦克不一定就得穿防御拿盾牌，队伍装备好坦克也可以穿输出拿双手武器

因有buff道具，没有打不过的boss……

基础随从命令

跟随

.npcbot command follow 让所有小伙伴跟随你只可在非战斗状态下使用

停止

.npcbot command stay 让所有小伙伴停留原地只可在非战斗状态下使用

刷新

.npcbot reset 刷新小伙伴 选定一个小伙伴后使用让其刷新，不选择任何目标则刷新全部小伙伴

跟随距离

可设置为0、15、30（近战）、75（远程）四种，在副本可减少引怪，也可以配合-跟随、停止-指令把卡位怪物引过来

.npcbot distance 数字 设定小伙伴跟随距离，方便给予其装备，分配其作战特色

默认值30，需要队长或团队领袖权限，也会影响全团他人召唤的小伙伴，值为0机器人不会主动打怪

其他命令

.npcbot remove 解散小伙伴选定一个小伙伴后使用将其与你解散，选择自己头像则让全部小伙伴与你解散 .npcbot remove all 解散所有小伙伴 .npcbot helper 打开小伙伴帮助菜单 .npcbot info 获得小伙伴详细信息需要选定一个小伙伴

召唤nbot团队

此命令每次会随机召唤nbot——除非精简至每职业10人，不然不能培养装备
